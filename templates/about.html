About MyToDo

MyToDo is a simple and efficient task management application built with Flask and SQLAlchemy. It helps you organize your daily tasks and keep track of what needs to be done.

Features

  • ✓ Create new todo items with title and description
  • ✓ View all your todos in one place
  • ✓ Update existing todos
  • ✓ Delete completed tasks

Technology Stack

Backend: Flask (Python Web Framework)
Database: SQLite with SQLAlchemy ORM
Frontend: Bootstrap 5.3
Template Engine: Jinja2
